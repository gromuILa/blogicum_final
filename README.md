# Блогикум — финальная версия

Блогикум — это социальная сеть для публикации статей и блогов.

## Функциональность

- Просмотр публикаций на главной странице, по категориям и по профилям пользователей
- Пагинация (10 публикаций на страницу)
- Регистрация и аутентификация пользователей
- Создание, редактирование и удаление публикаций (только автором)
- Прикрепление изображений к публикациям
- Комментарии к публикациям (редактирование и удаление автором)
- Кастомные страницы ошибок (403, 404, 500)
- Панель администратора для управления контентом
- Файловый бэкенд для отправки email

## Установка и запуск

### Требования
- Python 3.8+
- Django 3.2.16

### Инструкция

1. Клонировать репозиторий:
```bash
git clone https://github.com/gromuILa/blogicum_final.git
cd django_sprint4
```

2. Создать и активировать виртуальное окружение:
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. Установить зависимости:
```bash
pip install -r requirements.txt
```

4. Применить миграции:
```bash
python manage.py migrate
```

5. Запустить сервер:
```bash
python manage.py runserver
```

## Структура проекта

```
django_sprint4/
├── blogicum/
│   ├── blog/              # Приложение для публикаций
│   │   ├── models.py      # Модели Post, Category, Location, Comment
│   │   ├── views.py       # Представления
│   │   ├── forms.py       # Формы
│   │   ├── urls.py        # URL-маршруты
│   │   └── admin.py       # Админка
│   ├── pages/             # Приложение для статических страниц
│   ├── templates/         # HTML-шаблоны
│   │   ├── blog/          # Шаблоны публикаций
│   │   ├── pages/         # Статические страницы
│   │   ├── registration/  # Шаблоны аутентификации
│   │   └── includes/      # Переиспользуемые компоненты
│   ├── static/            # Статические файлы
│   └── blogicum/          # Настройки проекта
├── sent_emails/           # Папка для email (не в Git)
├── manage.py
└── requirements.txt
```

## Основные URL

- Главная: `/`
- Создание публикации: `/posts/create/`
- Профиль: `/profile/<username>/`
- Публикация: `/posts/<id>/`
- Категория: `/category/<slug>/`
- Авторизация: `/auth/`
- Админка: `/admin/`
